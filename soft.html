Software is a set of instructions, data or programs used to operate computers and execute specific tasks. Opposite of hardware. Software is often divided into categories. Application software refers to user-downloaded programs that fulfill a want or need. Examples of applications include office suites, database programs, web browsers, word processors, software development tools, image editors and communication platforms.

computer software can be divided into:

  • Application software
  • System software
  • operation system
  • device drivers
  • utilities
  • malware
